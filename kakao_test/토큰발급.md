카카오 API 토큰 발급은 보안상 **"사용자 인증 → 인가 코드 획득 → 토큰 교환"**이라는 3단계 과정을 거쳐야 합니다. 처음 한 번만 수동으로 진행하면, 그 이후부터는 앞서 드린 코드가 `refresh_token`을 사용해 알아서 관리할 것입니다.

차근차근 따라 해보세요!

---

### 1단계: 인가 코드(Authorization Code) 받기

먼저 브라우저(크롬 등) 주소창에 아래 주소를 편집해서 입력해야 합니다.

```text
https://kauth.kakao.com/oauth/authorize?client_id={내_REST_API_키}&redirect_uri={내_Redirect_URI}&response_type=code&scope=talk_message

```

* **{내_REST_API_키}**: 카카오 개발자 센터 > 내 애플리케이션 > 앱 설정 > **REST API 키**를 넣으세요.
* **{내_Redirect_URI}**: 플랫폼 설정에서 등록한 주소(예: `https://localhost:3000`)를 넣으세요.
* **scope=talk_message**: 메시지 전송 권한을 요청하는 옵션입니다.

입력 후 엔터를 치면 카카오 로그인 화면이 나옵니다. 로그인을 마치면 페이지는 오류가 날 수 있지만(localhost인 경우), **주소창**을 확인해 보세요!
`https://localhost:3000/?code=이부분이_인가_코드입니다...` 처럼 `code=` 뒤에 아주 긴 문자열이 생겼을 겁니다. 이 **인가 코드**를 복사하세요.

---

### 2단계: 인가 코드로 토큰(Access/Refresh) 발급받기

이제 이 인가 코드를 카카오 서버에 주고 실제 토큰으로 바꿔야 합니다. 아래 파이썬 코드를 내 컴퓨터에서 **딱 한 번만** 실행하세요.

```python
import requests

# 본인의 정보로 수정하세요
rest_api_key = '여기에_REST_API_키_입력'
redirect_uri = 'https://localhost:3000' # 등록한 주소
authorize_code = '여기에_아까_복사한_인가_코드_입력'

url = "https://kauth.kakao.com/oauth/token"
data = {
    "grant_type": "authorization_code",
    "client_id": rest_api_key,
    "redirect_uri": redirect_uri,
    "code": authorize_code,
}

response = requests.post(url, data=data)
tokens = response.json()

if "access_token" in tokens:
    print("발급 성공!")
    print(f"Access Token: {tokens['access_token']}")
    print(f"Refresh Token: {tokens['refresh_token']}")
else:
    print(f"발급 실패: {tokens}")

```

### 3단계: GitHub Secrets에 등록하기

출력된 결과값 중에서 **`refresh_token`**이 가장 중요합니다.

1. GitHub 레포지토리로 이동합니다.
2. `Settings` > `Secrets and variables` > `Actions` 클릭.
3. **New repository secret** 버튼을 눌러 다음 두 가지를 저장합니다.
* Name: `KAKAO_REST_API_KEY` / Value: (내 REST API 키)
* Name: `KAKAO_REFRESH_TOKEN` / Value: (방금 출력된 **refresh_token** 값)

### ⚠️ 주의사항 (동의항목 체크)

인가 코드를 받을 때 "메시지 전송" 권한 동의창이 안 뜬다면, 카카오 개발자 센터의 **[내 애플리케이션] > [제품 설정] > [카카오 로그인] > [동의항목]**에서 **'카카오톡 메시지 전송'**을 **'이용자 동의'** 상태로 변경해야 합니다.

이제 모든 준비가 끝났습니다! GitHub Actions가 정해진 시간에 `message.txt`를 읽어 자동으로 카톡을 보낼 거예요.

**혹시 `message.txt`에 매번 다른 내용을 담고 싶으신가요?** (예: 날씨 정보나 뉴스 크롤링 등) 필요하시다면 텍스트를 자동으로 업데이트하는 방법도 알려드릴 수 있습니다.
